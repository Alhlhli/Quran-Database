## 🔎 نظرة عامة

**قاعدة بيانات القرآن الكريم** صفحة ويب خفيفة وسريعة تعرض بيانات السور الـ114 في جدول
تفاعلي قابل للبحث والفرز والتصفية، مع صفحة إحصائيات بيانية مستقلة. التطبيق يعمل بالكامل
في المتصفح دون أي خادم خلفي، ويمكن تثبيته على الجوال كتطبيق (PWA) واستخدامه دون اتصال.

> **28 عمودًا من البيانات** لكل سورة، تشمل: ترتيب النزول، عدد الآيات/الكلمات/الحروف،
> الحروف المقطعة، السجدات، عدد مرات ذكر «الله»، أول/آخر كلمة، أكثر كلمة وحرف تكرارًا،
> الأسماء الحسنى، الأنبياء، الأرقام، الألوان، الحيوانات، أطول/أقصر آية، والمتوسطات.

## 🚀 التشغيل محليًا

> [!IMPORTANT]
> يجب فتح الصفحة عبر **خادم محلي** وليس بالنقر المزدوج على الملف،
> لأن المتصفح يمنع `fetch` لملف JSON عبر بروتوكول `file://`.

استنساخ المستودع:

```bash
git clone https://github.com/Alhlhli/Quran-Database.git
cd Quran-Database
```

ثم شغّل أي خادم ثابت — اختر أحد الخيارات:

```bash
# Python (مثبّت غالبًا)
python -m http.server 8753

# أو Node.js
npx serve .

# أو امتداد Live Server في VS Code
```

ثم افتح المتصفح على:

```
http://localhost:8753/index.html
```

## 🗂️ بنية المشروع

```
Quran-Database/
├── index.html        # الصفحة الرئيسية (الهيكل فقط)
├── stats.html        # صفحة الإحصائيات البيانية
├── style.css         # الأنماط — مقسّمة لأقسام + متغيرات ألوان
├── app.js            # منطق التطبيق — دوال صغيرة واضحة
├── stats.js          # حساب ورسم الإحصائيات
├── quran-data.json   # البيانات (صيغة عمودية مضغوطة)

├── sw.js             # Service Worker للعمل دون إنترنت
```

تقسيم الكود يحقّق: **تحميلًا أسرع**، و**تنظيمًا أفضل**، وإمكانية **تحديث البيانات
دون لمس صفحة الـ HTML**.

## 📦 صيغة البيانات

يستخدم المشروع صيغة **عمودية** مضغوطة (أصغر من مصفوفة الكائنات لأنها لا تكرّر المفاتيح):

```json
{
  "columns": ["id", "name", "number", "revelationOrder", "verses", "..."],
  "data": [
    [1, "الفاتحة", 1, 5, 7, 29, 143, "مكية", "..."],
    [2, "البقرة",  2, 87, 286, 6140, 26249, "مدنية", "..."]
  ]
}
```

يقوم التطبيق بتحويلها تلقائيًا إلى كائنات عبر `normalizeData()`، مع دعم الصيغة
القديمة `{ "surahs": [...] }` للتوافق.

## ⚙️ الإعدادات الشاملة

يفتح زر **الإعدادات** لوحة جانبية تتحكم في كل شيء، وتُحفظ التفضيلات في `localStorage`:

- **المظهر:** الوضع الليلي · تفعيل/إيقاف الحركات · كثافة الجدول · حجم الخط.
- **العرض:** عرض المفضلة فقط.
- **الأعمدة الظاهرة:** اختيار أهم الأعمدة لتقليل الازدحام (إظهار الكل / الافتراضي).
- **إعادة الضبط:** مسح المفضلة · استعادة جميع الإعدادات الافتراضية.

## 📱 دعم PWA (العمل دون إنترنت)

- التطبيق **قابل للتثبيت** على الجوال وسطح المكتب.
- يعمل **دون اتصال** بعد أول زيارة بفضل الـ Service Worker.

> عند تحديث الكود، ارفع رقم النسخة في `sw.js` (`const CACHE = 'quran-db-v2'`)
> ليحصل المستخدمون على آخر إصدار فورًا.

## 🛠️ التقنيات المستخدمة

- **HTML5 / CSS3** — متغيرات CSS، تصميم متجاوب، وضع ليلي.
- **JavaScript (Vanilla)** — بلا أطر، دوال صغيرة، `DocumentFragment` للأداء.
- **[Chart.js](https://www.chartjs.org/)** — الرسوم البيانية.
- **[SheetJS (xlsx)](https://sheetjs.com/)** — تصدير Excel.
- **[Font Awesome](https://fontawesome.com/)** — الأيقونات.
- **PWA** — `manifest.json` + Service Worker.
